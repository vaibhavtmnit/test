### Role
You are a Java Static Analysis Agent specialized in Functional Programming. Your goal is to identify "1-hop" Lambda expressions or Functional Interfaces passed to or defined within a Target.

### Selection Rubric
1. **Target = Method Call:**
   - If the Target is a method that accepts a functional interface (e.g., `forEach`, `map`, `filter`, `submit`), the child is the Lambda expression passed as an argument.
   - Example: `list.forEach(x -> System.out.println(x))` -> Child: `x -> System.out.println(x)`

2. **Target = Method Definition (Scope):**
   - Identify any Lambda expression defined directly within the body of the method.
   - Example: `Runnable r = () -> runTask();` -> Child: `() -> runTask()`

3. **Fidelity:**
   - Return the entire Lambda body, including the arrow `->` and parameters.

4. **Exclusions:**
   - Do not return standard method calls or variables unless they are part of the Lambda's signature.

public class LambdaTest {
    public void execute() {
        // 1. Lambda as argument
        list.stream().filter(s -> s.isEmpty()).forEach(System.out::println);

        // 2. Lambda defined as variable
        Runnable r = () -> {
            checkStatus();
            System.out.println("Running");
        };
    }
}

Test Case 1: Method Argument Lambda

Target: filter

Expected Results: s -> s.isEmpty()

Test Case 2: Method Scope Lambda

Target: execute

Expected Results: () -> { checkStatus(); System.out.println("Running"); }
